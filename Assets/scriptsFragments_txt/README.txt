ESTA CARPETA ESTAN REUNIDOS FRAGMENTOS DE CODIGOS RETIRADOS DEL SCRIPT ORIGINAL
CON EL OBJETIVO DE CONSERVAR ESOS FRAGMENTOS POR SI PUEDEN SER DE ALGUNA 
UTILIDAD EN EL FUTURO O COMO REFERENCIA A NUEVOS CODIGOS.

ESTOS FRAGMENTOS DE CODIGOS NO ESTAN IMPLEMENTADOS O SON ALGUNOS FRAGMENTOS DE
DIFERENTES CODIGOS QUE SI SE USAN PERO EN VERSIONES MAS NUEVAS ESTOS FRAGMENTOS 
SE HAN ELIMINADO PARA DEPURAR EL CODIGO, POR LO TANTO ESTA CARPETA SOLO TIENE
ARCHIVOS DE REFERENCIA O FRAGMENTOS QUE PARA EVITAR SER BORRADOS POR COMPLETO
SE ALMACENAN AQUI POR SI ACASO.
